### Phase 7.5 Step 2: Bartlett Test Implementation ✅ 完了
**分散の等質性検定 (正規分布仮定下での高精度)**
- [x] `bartlett_test(*groups)` メソッド実装

- [x] 合併分散S²ₚ計算とカイ二乗分布 χ²(k-1) によるp値
- [x] CLI統合: `'bartlett' => :run_bartlett` コマンド追加（24個目のサブコマンド）
